An open-source, high-performance show control and cueing engine built in Rust, this application is designed as a modern, lightweight alternative to QLab for Linux and Windows. It provides frame-accurate video playback, multi-channel audio routing, and complex integration cues without locking essential theatrical features behind restrictive paywalls. Leveraging a lock-free, concurrent architecture, the engine guarantees the ultra-low latency and rock-solid stability demanded by live, high-stakes productions.
